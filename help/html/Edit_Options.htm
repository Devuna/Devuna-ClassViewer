Edit Options

The Edit Options dialog is used to configure the appearance and behaviour of the program. The options, except for the Standard Equate Files and Additional Files List, are stored in the program INI file.

 

The program will first look in current directory for a file named ClassViewer.INI. If it finds a local ini file, it will use that one, otherwise it will default to the ini file located in the %windir folder. This is useful if the program is run from multiple folders to maintain several different database sets.

 

 

Include Standard Equate Files

Use the check boxes it select the standard equate files to be included in the scan. By default, the scan process only scans the class inc and clw files. There are several other standard files that contain items that may be of interest to the user and the check boxes provide a convenient way to specify the inclusions desired. Standard files that are not available in the %root%\libsrc folder are disabled.

 

Additional Files

Press the Additional Files button to display the Additional Files dialog which allows the user to include specific files.

 

Redirection File

The redirection file is the primary mechanism used by Class Viewer to determine what folders need to be scanned. The default value is the standard redirection file for the version of Clarion that has been selected. The user can override this file prior to scanning by pressing the ellipsis button to the right of the field. A file dialog is displayed to allow the user to select a redirection file.

 

 

Font

Use the ellipsis to the right of the field to select the font to be used by the application. Care should be taken not to choose too large of a font size as the screens may not be able to accommodate it.

 

Background

The drop list provides the following choices:

Color

Choose this option to select colors to be used for the application windows.

 

Color 1
This is the color used by the Main window.
 
Color 2
This is the color used by all windows except the Main window.
Wallpaper

Choose this option to select wallpapers to be used for the application windows.

 

Wallpaper 1
This is the wallpaper used by the Main window.
 
Wallpaper 2
This is the wallpaper used by all windows except the Main window.
None

Choose this option to disable the use of colors or wallpapers for the application windows.

 

DoubleClick Action

Select the default program action when a Tree View item is double clicked.

  • View      invoke the Ascii File Viewer

  • Edit      invoke the default Editor
     

    • Enable Balloon Tips

    Check this option to enable balloon style tooltips. You will need to restart the application in order for a change in this option to have effect.

     

    Some users have reported problems seeing tooltips when this option is enabled. If the program is not displaying tooltips, try turning this option off.

     

    Auto Expand

    Check this option to have the program automatically expand an item in the Tree View when it is selected in the Object List. If this option is unchecked, then the Tree View for the selected object will not be expanded.

     

    Show Sparse Trees

    Check this option to limit the tree to the selected class and its direct parents. Uncheck to have the tree display the class parents and all classes derived from the parent classes.

     

    Use html Help

    Check this option to use html style help. Uncheck to use Windows style help.

     

    Favorites Menu...

    Press this button to access the Favorites Menu dialog. The dialog is used to configure the Favorites Menu located on the Main window Tools Menu.

     

    XML Stylesheet

    Use the ellipsis button to the right of this field to select an XML stylesheet. If the field is not empty, the XML export options will place a <?xml-style ?> processing instruction near the beginning of the exported xml files. If you do not use style sheets to view the XML export files, then leave this field blank. To clear the field, remove the XmlStyleSheet= line from the abcview.ini file.

     

     

    Selected Back

    Use the ellipsis button to the right of the field to select the Selected Back color. This color is used in the application list boxes, the Ascii File Viewer, and in the refresh progress bar.

     

    Module Color

    Use the ellipsis button to the right of the field to select the Module color. This color is used to identify items that are declared within the class definition module.

     

    Private Color

    Use the ellipsis button to the right of the field to select the Private color. This color is used to identify items that have the private attribute.

     

    Protected Color

    Use the ellipsis button to the right of the field to select the Protected color. This color is used to identify items that have the protected attribute.

     

    Virtual Color

    Use the ellipsis button to the right of the field to select the Virtual color. This color is used to identify items that have the virtual attribute.

     

    Hyperlink Color

    Use the ellipsis button to the right of the field to select the Hyperlink color. This color is used (in addition to the underline font) to identify references to objects that can be hyperlinked. This color will not override the Private or Protected attribute colors.

     

    Note Color

    Use the ellipsis button to the right of the field to select the Note background color. This color is used as the background color when viewing notes.

     

    Highlight Color 1

    Use the ellipsis button to the right of the field to select the Ascii File Viewer Highlight Color 1. This color is used by the Ascii File Viewer to identify the source line that was jumped to when the Ascii File Viewer was invoked. The background color used with the highlight color is COLOR:BTNFACE.

     

    Highlight Color 2

    Use the ellipsis button to the right of the field to select the Ascii File Viewer Highlight Color 2. This color is used by the Ascii File Viewer to identify the lines that contain references to item highlighted in the Tree View when the Ascii File Viewer was invoked. The background color used with the highlight color is COLOR:BTNFACE.

     

     

    Use Windows Associations

    Check this option if you want to use Windows Associations when initiating an edit process for a file.

     

    Command Line

    Enter the command line to be used when initiating an edit process for a file. Use the ellipsis button to the right of the field to locate the editor program.

    Class Include File Templates

    Press this button to access the inc Template file registry. inc Templates are simple text files that are used by the Add Class Wizard to generate class include (inc) files. Before they can be used, templates must be added to the template registry.

     

    Class Source File Templates

    Press this button to access the clw template file registry. clw templates are simple text files that are used by the Add Class Wizard to generate class source (clw) files. Before they can be used, templates must be added to the template registry.

     

    Wrapper Generator Templates

    Press this button to access the wrapper generator template registry. Wrapper generator templates are simple text files that are used by the Wrapper Generator to generate template (tpl) files.

    Before they can be used, templates must be added to the template registry.

     

     

    The Viewer Folder contains one folder for Default settings and then a folder for each lexer style used by the scintilla control. The options for each folder are the same.

     

    Font

    Use the ellipsis to the right of the field to select the font to be used by the style identified by the current tab.

     

    Fore Color

    Use the ellipsis button to the right of the field to select the Foreground color.

     

    Back Color

    Use the ellipsis button to the right of the field to select the Background color.

     

    Case

    Select the case to be used by the style identified by the current tab.

     

    Options

    Select athe additional options to be used by the style identified by the current tab.

     

    EOL Filled
    If the last character in the line has a style with this attribute set, the remainder of the line up to the right edge of the window is filled with the background colour set for the last character. This is useful when a document contains embedded sections in another language such as html pages with embedded JavaScript. By setting eolFilled to true and a consistent background colour (different from the background colour set for the html styles) to all JavaScript styles then JavaScript sections will be easily distinguished from html.
    Visible
    Text is normally visible. However, you can completely hide it by giving it a style with the visible set to 0. This could be used to hide comments for example.
    Hot Spot
    This style is used to mark ranges of text that can detect mouse clicks. The cursor changes to a hand over hotspots, and the foreground, and background colours may change and an underline appear to indicate that these areas are sensitive to clicking. ClassViewer uses this to access the Clarion help file when the text is clicked.

     

    Apply Defaults

    This button only appears when the Default Tab is selected. Press this button to display a message box that will prompt for the Default characteristics to apply. The choices are:

     

    Font            Apply the default font to all styles.
    Background      Apply the default back color to all styles.
    All            Apply all the default characteristics to all styles.
    Cancel            Do not apply any default characteristics and return to the Edit Options
     

    Clarion Editor

    This button only appears when the Default Folder is selected. Press this button to display a message box that will prompt for the Clarion Editor colors to load. The choices are:

     

    Current            Load and apply the current Clarion Editor colors from C60EDT.INI.
    Default            Load and apply the default Clarion Editor colors from C60EDT.INI.
    Cancel            Do not load and apply the Clarion Editor colors and return to the Edit Options.

     

    OK

    Press the OK Button to save your changes and return to the Main window.

     

    Cancel

    Press the Cancel Button to discard any changes you have made and return to the Main window.
